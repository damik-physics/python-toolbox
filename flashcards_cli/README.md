# Flashcards CLI

A small command line tool for creating flashcards, storing them in JSON, and running simple quizzes. Intended as a lightweight study helper.

## Features

- Create cards
- Remove cards
- View stored cards
- Quiz mode
- JSON based storage

## Installation

Clone the repository:

```bash
git clone https://github.com/damik-physics/python-toolbox.git
cd python-toolbox/flashcards_cli
```

Run the tool with Python:

```bash
python3 cli.py
```

## File Structure

```text
flashcards_cli/
    cli.py          Command line interface
    storage.py      JSON load and save logic
    README.md
```

## Requirements

Python 3.10 or newer.
